** Library Book Management System**
The Book Management System is a Java Dynamic Web Application developed using Servlets, JSP, JDBC, and Oracle Database.
This application allows users to add and manage book details such as ISBN, Book Name, Book Type, Author Name, and Book Cost.

The project demonstrates:

Java Servlet development
JDBC connectivity with Oracle 21c XE
MVC architecture
Session handling
Form validation
Database integration

Features

Add new book
View book details
Server-side validation
Database storage using Oracle
Dynamic dropdown (Author & Book Type)
Invalid data handling

Technologies Used

Java
Servlet
JDBC
Oracle Database
Apache Tomcat Server
HTML
Eclipse IDE (Dynamic Web Project)
